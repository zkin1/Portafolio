Proyectos

Proyecto 1

He desarrollado un software de gestión de arriendos de vehículos utilizando Java y MySQL. Esta aplicación permite registrar diferentes tipos de vehículos, incluyendo autos y camionetas, con el propósito de facilitar su arriendo. La aplicación cuenta con una base de datos que almacena información detallada sobre los vehículos, incluyendo sus características específicas y su estado de arriendo, lo que permite llevar un registro preciso de quién está arrendando cada vehículo en un momento dado, así como también quiénes son los propietarios de los mismos. Esta solución proporciona una plataforma intuitiva y eficiente para la gestión de arriendos de vehículos, mejorando la organización y facilitando la administración de flotas de automóviles y camionetas.
